# 📊 Olist Ecommerce Analysis — Power BI

## 📌 Sobre o projeto

Projeto de análise de dados de e-commerce utilizando a base pública da Olist.

O objetivo foi analisar vendas, clientes, pedidos e operações para gerar insights através de um dashboard interativo desenvolvido no Power BI.

---

## 🛠️ Ferramentas utilizadas

- Power BI
- SQL
- SQLite
- Excel / CSV
- GitHub

## 💡 Principais insights

- Identificação da evolução das vendas ao longo do período analisado.
- Análise do comportamento dos clientes e volume de pedidos.
- Avaliação dos produtos com maior participação nas vendas.
- Monitoramento de indicadores operacionais relacionados aos pedidos.

# 🔎 Principais análises realizadas

- Evolução do faturamento e vendas
- Volume de pedidos
- Perfil dos clientes
- Produtos mais vendidos
- Análise operacional dos pedidos
- Indicadores de desempenho

---
# 🎯 Objetivo

Transformar dados brutos de e-commerce em informações estratégicas para apoiar tomada de decisão.
